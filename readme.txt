== Description ==


Wtyczka generuje pliki XML, umożliwiający integracje sklepu z Slowfashion.pl.


== Installation ==




= WYMAGANIA =


Zgłoś swój sklep na https://jestemslow.pl


= INSTALACJA MODUŁU =


* Aktywuj wtyczkę
* Pliki XML są generowane co trzy godziny. Aktualny status i adres URL do pliku XML można znaleźć na stronie ustawień "WooCommerce => Woocommerce Feed Slowfashion"
* Musisz mieć zainstalowany i włączony moduł openssl na twoim serwerze, aby poprawnie zaktualizować kategorie. Może być również konieczne włączenie opcji "have allow_url_fopen" i "allow_url_include".
